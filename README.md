# 项目简介

## 核心功能
- 基于YOLOv8的玉米叶片病害检测，支持6类病害识别：草地贪夜蛾卵、健康叶片、草地贪夜蛾危害后、叶枯病、普通锈病、灰斑病
- 支持多种输入源检测：单张图片、图片文件夹、视频文件、本地摄像头
- 提供自定义参数功能：置信度阈值、IOU阈值调整
- 支持模型选择与结果显示控制：可选择显示标签和原图
- 批量检测功能：可批量处理图片并查看所有检测结果
- 检测结果保存功能：支持保存检测结果

## 适用场景
- 农业病害监测与研究
- 玉米种植健康管理
- 农业科研数据分析
- 农作物病害早期预警系统

## 技术特点
- 技术栈：Python3.9、OpenCV、PyQt5、Ultralytics、PyTorch1.9
- 开发环境：PyCharm+Anaconda 或 VSCode+Anaconda
- 数据集规模：4000张标注图片
- 已完成模型训练，提供完整训练结果
- 配置成功环境后可直接使用
